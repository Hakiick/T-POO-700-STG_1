# T-POO-700 — TimeManager

## Project overview
Application de gestion du temps (time tracking) avec dashboard, gestion d'équipes, et administration.

## Stack technique
- **Framework** : Phoenix (Elixir) backend API + Vue.js 3 frontend SPA
- **Styling** : Tailwind CSS
- **Tests** : Vitest (frontend)
- **Linter** : ESLint
- **Database** : PostgreSQL

## Architecture
- `backend/` — API Phoenix/Elixir
- `frontend/` — SPA Vue.js 3 + Vite + Tailwind
- `docker-compose.yml` — Orchestration des services
